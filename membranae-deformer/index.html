Membranae Deformer

Membranae is a deformer for Autodesk Maya that allow to constraint a polygon mesh to a nurbs mesh but avoiding any squash and streatch when the nurbs surface change shape.

The result is a polygon mesh that "float" on the nurbs making this deformer perfect for bird's wing setups
